## What this project does

Website research assistant using RAG

Ask questions from website content using Retrieval-Augmented Generation (RAG) built with LangChain and Streamlit.





## Features



✔ URL ingestion

✔ Web scraping 

✔ Document chunking

✔ Embeddings

✔ Vector database search

✔ Retrieval QA

✔ Source citations

✔ Retrieval transparency with similarity score

✔ Streamlit UI

✔ Session state handling

✔ Error handling
